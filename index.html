مادة الشرح والأساسيات التقنية

تم تجميع هذا الشرح الفني من المنهج التدريبي المعتمد لشركة المصرية للاتصالات

🛠️ البنية الفيزيائية ومسار التوصيل (Physical Structure)

مسار التوصيل الفيزيائي للخط الأرضي: تبدأ رحلة إشارة الحرارة بالسنترال بخطة مسبقة لدمج البيانات والصوت عبر الخطوات التنازلية التالية:

  • Zero Room: منبع ومولد إشارة الحرارة التناظرية في السنترال.
  • MSAN Cabin: كابينة ألياف الفايبر الذكية التي تدمج خدمة الإنترنت فائق السرعة مع خط الصوت.
  • Cabins (الكابينات): كبائن نحاسية فرعية لتوزيع وإتاحة وتوصيل الأسلاك بالشوارع.
  • Box (البوكس): نقطة التوزيع المعلقة بواجهات المباني والمنشآت.
  • CPE (الراوتر): المعدات والمنفذ الأخير المستلم للإشارة بمنزل المشترك.

التوصيلات عند العميل:

  • Splitter: جهاز بيفصل الإنترنت عن الحرارة، يضم 3 فتحات: Line (السلك الرئيسي من الشارع)، Modem (إلى الراوتر)، Phone (بالتليفون الأرضي).
  • Built-in Splitter: مدمج داخل فيشة الحائط ولا يُنصح بالاعتماد عليه.
  • Blank Splitter: خالي من الكتابة، ويُنصح باستبداله تجنباً للأخطاء.
  • Rosetta: أداة لحل مشكلات اللحام والقطع لتقليل المقاومة والتشويش وبها مخرج RJ11.
  • DSL Filter: أداة تعزل إشارة الإنترنت لتنقية خط الهاتف الأرضي من التشويش.
  • PBX: سنترال داخلي للحرارة للشركات، ويلزم توصيله بعد الـ Splitter.
📌 ملاحظة فنية: لحام الأسلاك الرديء أو استخدام الـ RJ11 Joint بكثرة يزيد من الـ Attenuation ويولد أخطاء CRC تقطع الخدمة تماماً.

⚙️ البنية المنطقية وإصدار العناوين (Logical Structure)

سحب الـ WAN IP للراوتر: لا تقتصر العملية الفنية على وجود حرارة بالخط، بل تتطلب دورة مصادقة ديناميكية (Logical Flow) للتحقق من هوية المشترك:

  • Gateway IP: بوابة الدخول لصفحة الراوتر (غالباً 192.168.1.1).
  • Private IP / LAN IP: عنوان محلي للأجهزة داخل الشبكة المغلقة.
  • Public IP / WAN IP: العنوان العام الممنوح من الـ ISP لتصفح الإنترنت الخارجي.
  • DNS (Domain Name System): يقوم بترجمة عناوين المواقع النصية لـ IPs تفهمها الشبكة.
  • DHCP: المسؤول عن توزيع الـ Private IP تلقائياً للأجهزة.
  • NAT (Network Address Translation): المسؤول عن تخرج الأجهزة من الشبكة الداخلية (Private) للشبكة الخارجية (Public).
  • APIPA (Automatic Private IP Addressing): نطاق IP طارئ (169.254.0.1 إلى 169.254.255.254) يأخذه الجهاز تلقائياً عند فشل الحصول على عنوان من الـ DHCP، وحله عمل Reset & configuration للراوتر.

معاملات التهيئة (CPE Configuration):

  • 🔹 ADSL Parameters: VPI/VCI: 0/35 | Mode: Routing | Encapsulation: PPPOE | Multiplex: LLC
  • 🔹 VDSL Parameters: XDSL Transfer Mode: PTM | Link Type: PPP | PPP Transfer Type: PPPOE
  • 🔹 MTU (الحجم الأقصى لوحدة النقل): القيمة الافتراضية 1492، وقيم بديلة: 1460 - 1420 في حالات التشويش.
  • 🔹 Proxy: سيرفر أو تطبيق وسيط بين شبكة الـ LAN وشبكة الـ WAN لتخطي مشكلات تصفح المواقع.
🔑 مرحلة الربط النهائي: يقوم الـ BRAS بإصدار الـ WAN IP للعميل بعد التحقق من الـ RADIUS & NAS port ID ومطابقة البورت الفعلي (Card & Port)، وتطبيق الـ Profile المعتمد عبر قاعدة الـ Broad-Hop.

📖 القاموس المعتمد للاتصالات (WE Brand Technical Dictionary)

ابحث بمرونة تامة عن أي اختصار مستخدم ببيئة عمل الاتصالات ومراكز الاتصال

ACW After Call Work

الوقت المستغرق بعد إنهاء المكالمة لتسجيل الملاحظات، فتح التذاكر، وتحديث السيستم.

ADSL Asymmetric Digital Subscriber Line

خط المشترك الرقمي غير المتماثل: التكنولوجيا النحاسية الكلاسيكية التي تقدم سرعة تحميل أعلى من سرعة الرفع.

AHT Average Handling Time

متوسط وقت مناولة المكالمة: الوقت الإجمالي المستغرق لمعالجة الشكوى (شامل التحدث والانتظار والتوثيق).

ATT Average Talk Time

متوسط وقت التحدث الفعلي والمباشر مع المشترك أثناء المكالمة.

ATT / LAT Line Attenuation / Attenuation

وهن أو خفات الخط: يمثل مقدار الفقد الحادث في الإشارة الكهربائية نتيجة طول السلك والبعد عن الكابينة، كلما قل الرقم زادت الكفاءة والسرعة.

BSS Business Support System

نظام دعم الأعمال: الأنظمة والبرامج المخصصة لإدارة العمليات التجارية والمالية واشتراكات العملاء.

CCA Customer Care Agent / Call Center Agent

موظف خدمة العملاء / مركز الاتصال: المسؤول عن استقبال مكالمات ودعم المشتركين.

CPE Customer Premises Equipment

معدات مواقع العملاء: أي جهاز طرفي يمتلكه المشترك بالمنزل لاستقبال وتمرير الخدمة مثل الراوتر أو المودم.

CRC Cyclic Redundancy Check Errors

أخطاء فحص التكرار الدوري: عدد حزم البيانات التالفة بالكامل بسبب التشويش أو اللحامات الرديئة والرطوبة بالسلك، وتسبب تقطيعاً مباشراً بالخدمة.

VCI Virtual Channel Identifier

معرف القناة الافتراضية: القيمة الرقمية المكملة للـ VPI لضمان وصول مسار خدمة الإنترنت بالراوتر فنيًا (القيمة الثابتة والمعتمدة بمصر هي 35).

VDSL Very High Speed Digital Subscriber Line

خط المشترك الرقمي عالي السرعة جداً: التقنية الحديثة المعتمدة عبر كبائن الـ MSAN لتقديم سرعات إنترنت عالية تصل إلى 100 ميجا.

VLAN ID Virtual Local Area Network Identifier

معرف الشبكة المحلية الافتراضية: قيمة رقمية مخصصة لفصل وعزل خدمات الاتصالات المختلفة (إنترنت، خط صوتي، IPTV) لعملاء الـ VDSL وشبكات الفايبر.

VOIP Voice over Internet Protocol

نقل الصوت عبر بروتوكول الإنترنت: تشغيل وإدارة خدمات الهاتف الأرضي والاتصالات الصوتية رقمياً عبر الراوتر والشبكة مباشرة بدلاً من السنترال التقليدي القديم.

VPI Virtual Path Identifier

معرف المسار الافتراضي: القيمة الرقمية البرمجية المستخدمة لتوجيه حزم البيانات لشبكات الـ ADSL (القيمة الثابتة والمعتمدة بمصر هي 0).

VPN Virtual Private Network

الشبكة الافتراضية الخاصة: توفير نفق افتراضي مشفر وآمن بالكامل لتصفح الويب وتغيير الموقع الجغرافي وحفظ خصوصية البيانات الفردية.

WAN Wide Area Network

الشبكة الواسعة الخرجية: منفذ الراوتر الخارجي المسؤول عن استقبال وضخ شبكة الإنترنت العامة القادمة من مزود الخدمة الرئيسي للراوتر.

WLAN Wireless Local Area Network

الشبكة المحلية اللاسلكية: تقنية الربط اللاسلكي (Wi-Fi) التي تسمح للأجهزة المحمولة بالاتصال بالشبكة المحلية بالراوتر بدون أسلاك.

🌐 البروتوكولات والعناوين (Network Protocols)

Gateway IP (بوابة العبور):

هو الـ IP الخاص بالدخول لصفحة إعدادات الراوتر، ويكتب في ملصق ضهر الراوتر ويأتي بقيمة افتراضية (غالباً 192.168.1.1).

APIPA (نطاق الطوارئ):

عنوان (يبدأ من 169.254.0.1 لـ 169.254.255.254) يحصل عليه الحاسوب تلقائياً في حالة تعطل أو فقدان الاتصال بخادم DHCP بالراوتر.

NAT (ترجمة العناوين):

المسؤول عن تحويل الـ Private IPs للأجهزة المحلية إلى عنوان Public IP خارجي لتتمكن من التصفح بشكل آمن.

DHCP & DNS:

DHCP: يوزع IPs داخلية للأجهزة بشكل تلقائي.
DNS: المترجم الذكي الذي يحول الكلمات النصية (مثل google.com) لـ IP يفهمه السنترال.

⚙️ قيم ومعاملات تهيئة الراوتر (CPE Parameters)

إعدادات الـ ADSL الأساسية:

  • VPI/VCI: بقيم 0/35 المعتمدة بمصر.
  • Mode: Routing (نظام التوجيه التلقائي).
  • Encapsulation: PPPOE.
  • Multiplex: LLC.

إعدادات الـ VDSL فائقة السرعة:

  • XDSL Transfer Mode: PTM.
  • Link Type: PPP.
  • PPP Transfer Type: PPPOE.
الـ MTU وقيم التحكم في حجم البيانات:

يمثل الحجم الأقصى لنقل حزمة البيانات، وقيمته الافتراضية بمصر هي 1492، ويمكن خفضها لـ 1460 أو 1420 في حالات البطء وتشويش الخطوط.
